# Rust Workspace

This workspace owns the compiler/export side of the scaffold.

## Crates

- `arbx_geo` — basic coordinate and bounding types
- `arbx_pipeline` — project-owned domain and stage contracts
- `arbx_roblox_export` — manifest types and exporter helpers
- `arbx_cli` — developer CLI for samples and future validation

## Current goal

Keep dependencies intentionally light until the manifest contract settles.
